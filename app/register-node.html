Don't have a Cardano wallet? Here's how → ▼
To receive earnings from your node, you need a Cardano wallet address.

EASIEST OPTIONS (free):

1. Eternl Wallet (browser extension)
  → eternl.io → Install → Create wallet
  → Copy your address (starts with addr_test1 for testnet)

2. Lace Wallet (browser extension)
  → lace.io → Install → Create wallet
  → Switch to Preview Testnet in settings
  → Copy your address

GET FREE TESTNET ADA:

After creating your wallet, get free tADA:
  → faucet.preview.cardano.org
  → Paste your address → Request funds
  → Wait 1-2 minutes
